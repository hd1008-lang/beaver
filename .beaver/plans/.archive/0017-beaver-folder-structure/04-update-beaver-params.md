## Goal

Update `test/helpers/beaver-params.ts` to set `baseDir: '.beaver'` for beaver's harness and update REGEN_PREFIXES/REGEN_FILES to reflect `.beaver/` paths.

## Steps

- [x] **Edit `test/helpers/beaver-params.ts`** — add baseDir field to beaverParams:
  - Add after `harness: 'both'` line:
    ```typescript
    baseDir: '.beaver',
    ```
  - This instructs buildHarnessFileMap to render all {{baseDir}}/... paths with `.beaver/` prefix

- [x] **Update REGEN_PREFIXES** (lines ~49-55):
  - Current (tool-discovery paths):
    ```typescript

      'scripts/',
      '.claude/scripts/',
      '.claude/agents/',
      '.claude/skills/',
      '.codex/',
    ] as const;
    ```
  - New (knowledge-base paths now prefixed with `.beaver/`):

    ```typescript

      '.beaver/AGENTS.md',        // moves to .beaver/

      '.beaver/plans/README.md',  // moves to .beaver/
      '.beaver/backlog/README.md', // moves to .beaver/
    ```
